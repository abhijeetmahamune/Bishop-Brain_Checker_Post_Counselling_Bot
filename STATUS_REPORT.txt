TROUBLESHOOTING
──────────────────────────────────────────────────────────────────────────────

If upload still fails:

1. BACKEND NOT STARTING?
   - Check Python is installed: python --version
   - Install dependencies: pip install -r backend/requirements.txt
   - Look for error messages in terminal
   - Make sure .env file exists with API key

2. BACKEND RUNNING BUT UPLOAD FAILS?
   - Open browser DevTools: Press F12, go to Console tab
   - Check for specific error messages
   - Try uploading test_sample.pdf
   - Check backend logs in terminal
   - Run: python test_upload.py

3. CORS ERRORS?
   - Backend has CORS enabled for all origins
   - Should not be an issue
   - Check browser console for exact error

4. API KEY INVALID?
   - Check .env file has valid OPENROUTER_API_KEY
   - Get key from https://openrouter.ai/keys
   - Restart backend after updating .env
